The construction of the multiplication table relies upon four values: the minimum row value, the maximum row value, the minimum column value, and the maximum column value. It is important to make sure the minimum values are smaller than their respective maxmimum values. If there is a case where the user enters a larger minimum value than the maximum value counterpart, these values will be switched; the user will be notified with a message.

Please make sure not to enter letters or special characters. The only input allowed is the use of whole numbers between -50 and 50 inclusively. One may use the slider or the textbox to input their data.
